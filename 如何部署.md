# 🚀 部署到云端 - 操作指南

让你的 AI 聊天机器人上线，别人可以通过网址访问！

---

## 🎯 最快方案：Render（5分钟搞定）

Render 有免费套餐，支持从 GitHub 自动部署。

### 第一步：推送代码到 GitHub

1. 打开 https://github.com/login 登录你的 GitHub 账号
2. 点击右上角 **"+"** → **"New repository"**
3. 仓库名填：`chat-agent-app`
4. 选 **Public**（公开）
5. 点击 **"Create repository"**
6. 在仓库页面记下你的用户名（URL 里的就是）

然后在你的电脑上双击运行 `deploy.bat`，按提示输入用户名和 Token。

> **如何获取 Token？**
> - 打开 https://github.com/settings/tokens
> - 点击 **"Generate new token (classic)"**
> - 勾选 `repo` 权限
> - 生成后复制保存（只显示一次！）

### 第二步：在 Render 部署

1. 打开 https://dashboard.render.com 注册/登录（可以用 GitHub 登录）
2. 点击 **"New +"** → **"Web Service"**
3. 选择你刚创建的 `chat-agent-app` 仓库
4. Render 会自动读取 `render.yaml` 配置，确认以下设置：
   - **Build Command**: `npm install && npm run build`
   - **Start Command**: `NODE_ENV=production npm start`
5. 在 **Environment Variables** 中添加：
   - `CODEBUDDY_API_KEY` = `你的API密钥`
   - `NODE_ENV` = `production`
6. 点击 **"Apply"** 开始部署

### 第三步：获得公网地址 🎉

部署完成后（约2-3分钟），Render 会给你一个地址：
```
https://chat-agent-app-xxxx.onrender.com
```

这个地址任何人都可以访问！

## 📋 备选方案

### 方案 B：Railway（也有免费额度）

1. 双击 `railway-login.bat` 登录 Railway
2. 双击 `railway-deploy.bat` 一键部署
3. 输入 CODEBUDDY_API_KEY 环境变量

### 方案 C：如果不想用 GitHub

1. 在 Render 选择 "Deploy from ZIP"
2. 上传 `chat-agent-app-deploy.zip`
3. 配置 Build/Start 命令和环境变量
